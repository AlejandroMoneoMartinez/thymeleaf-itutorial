Thymeleaf tutorial - Exercise 11: links

This exercise introduces concepts from chapter 4 - section 4 of the Using Thymeleaf manual

  1. Open the WEB-INF/templates/exercise11/answer.html template file with your code editor.
  2. Modify the two links in order to point to /exercise11/product.html URL with the request parameter action. The value for action parameter must be view for the first link and edit for the second.
  3. Check your answer, clicking on the links to ensure they work alright: Execute it.
  4. Now modify the template so that when open statically those links point to their corresponding static HTML documents, this is viewProduct.html and editProduct.html.
  5. Check your answer again: See it statically | Execute it.
  6. Finally, you can compare your answer with our proposed solution: Execute it | See the code | See it statically.
