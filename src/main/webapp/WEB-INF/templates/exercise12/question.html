Thymeleaf tutorial - Exercise 12: forms

This exercise introduces concepts from chapter 4 - section 3 of the Using Thymeleaf manual and also chapter 6 of the Thymeleaf + Spring 3 manual.

  1. Open the WEB-INF/templates/exercise12/answer.html template with some code editor
  2. A bean of class Customer has been set into the context model with variable name customer. This is a complex bean, containing properties of the Gender and PaymentMethod enum types.
  3. Modify the template in order to bind bean fields into form fields. You have to use th:object and th:field attributes.
    As this exercise is more complex than previous ones, some hints have been written as HTML comments in the answer.html template.
  4. Check your answer, clicking on the submit button in order to check that the form works correctly: Execute it | See it statically.
  5. Finally, you can compare your answer with our proposed solution: Execute it | See the code | See it statically.
