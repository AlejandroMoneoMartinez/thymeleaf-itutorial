Thymeleaf tutorial - Exercise 2: simple formatting

This exercise introduces concepts from chapter 16 of the Using Thymeleaf manual

  1. Open the WEB-INF/templates/exercise02/answer.html template file with your code editor.
  2. Modify the template in order to show bean values as you did in exercise 1. A bean of class Product has been already set into the context model with variable name product.
  3. Now let's add some formatting to those Integer and Date properties. For this, have a look at the definition of the #dates and #numbers utility objects at the Expression Utility Objects Appendix of the Using Thymeleaf tutorial.
  4. Check your answer: Execute it.
  5. Now modify the template in order to obtain a static prototype that makes sense (for example, by substituting all those XXX by some prototyping data).
  6. Check your answer again: See it statically | Execute it.
  7. Finally, you can compare your answer with our proposed solution: Execute it | See the code | See it statically.
