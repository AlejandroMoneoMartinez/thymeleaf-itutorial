The Thymeleaf Interactive Tutorial

Welcome to the Thymeleaf Interactive Tutorial. Thymeleaf version: 2.0.15

Description

  • This tutorial will guide you through the most important aspects of the Thymeleaf Standard Dialects.
  • You will need:
    • A browser – this one will do! ;-)
    • A text editor for editing files in WEB-INF/templates, directly on the webapps folder of your web server, while it is running. Preferably, pick an editor with Java & HTML syntax highlight, like Notepad++, Textpad or gedit.
    • A PDF file reader in order to open the Thymeleaf tutorial guides, if you need them.
  • At some points during this tutorial you will be opening files in your browser in a static way, this is, without Thymeleaf executing them so that you can experience the Natural Template advantage. You will be able to notice that you are watching a static file because background will be grey instead of pale yellow. Usually a Back button will get you out of the static display, but you can also go back to the executed mode simply by removing the "/static" part of the URL.
