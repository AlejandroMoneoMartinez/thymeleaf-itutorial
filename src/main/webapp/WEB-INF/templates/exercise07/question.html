Thymeleaf tutorial - Exercise 7: iteration stats

This exercise introduces concepts from chapter 6 - section 2 of the Using Thymeleaf manual

  1. Open the WEB-INF/templates/exercise07/answer.html template file with your code editor.
  2. Modify the template in order to show the row index in the first column. Tip: use the iteration status variable from your th:each.
  3. Check your answer: Execute it.
  4. Now modify the template in order to obtain a more real-looking static prototype by adding some additional rows to the table (these rows should be removed when Thymeleaf executes).
  5. Check your answer again: See it statically | Execute it.
  6. Finally, you can compare your answer with our proposed solution: Execute it | See the code | See it statically.
