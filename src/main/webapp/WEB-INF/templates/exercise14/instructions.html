This exercise introduces concepts from chapter 8 of the Using Thymeleaf manual.

  1. Open the answer1.html, answer2.html and answer3.html template files from the WEB-INF/templates/exercise14/ folder with your code editor.
  2. Set /logo.png as the source for the img logo element.
  3. Turn the div element in answer1.html into a fragment by using th:fragment, and include it into the other templates just below the Product List header. Do this in two different ways:
    • Into answer2.html, using its fragment name (standard Thymeleaf-to-Thymeleaf) inclusion mechanism
    • Into answer3.html, using the DOM selector expression //div[@id='bannerElement'].
    When specifying your template inclusions, remember that template roots must start with /exercise14, like "/exercise14/answer1.html".
  4. Check your answer:
    • answer1.html: Execute it | See it statically.
    • answer2.html: Execute it | See it statically.
    • answer3.html: Execute it | See it statically.
  5. Finally, you can compare your answer with our proposed solution:
    • solution1.html: Execute it | See the code | See it statically.
    • solution2.html: Execute it | See the code | See it statically.
    • solution3.html: Execute it | See the code | See it statically.
  6. You can check the proposed solution by pressing the Show solution button.
