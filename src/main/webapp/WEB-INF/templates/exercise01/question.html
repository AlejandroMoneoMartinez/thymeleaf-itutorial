Thymeleaf tutorial - Exercise 1: bean values

This exercise introduces concepts from chapter 4 of the Using Thymeleaf manual

  1. Open the WEB-INF/templates/exercise01/answer.html template file with your code editor.
  2. Modify the template in order to show bean values. A bean of class Product has been already set into the context model with variable name product.
  3. Check your answer: Execute it.
  4. Now modify the template in order to obtain a static prototype that makes sense (for example, by substituting all those XXX by some prototyping data).
  5. Check your answer again: See it statically | Execute it.
  6. Finally, you can compare your answer with our proposed solution: Execute it | See the code | See it statically.
